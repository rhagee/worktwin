---
name: worktwin-help
description: List every worktwin command installed on this machine, with the arguments each one takes and a short description. Generated by reading the installed SKILL.md files so the list always matches reality. Cheap, deterministic, safe to spam.
disable-model-invocation: true
allowed-tools: Bash(ls *), Bash(cat *), Bash(grep *), Bash(sed *), Bash(git *), Bash(basename *), Bash(echo *), Bash(test *)
---

The block below is the entire help. Show it to the user verbatim. Do not paraphrase, do not summarise, do not add commentary before or after.

```!
SKILLS_DIR=""
if [ -d "$HOME/.claude/skills" ]; then
  SKILLS_DIR="$HOME/.claude/skills"
else
  REPO=$(git rev-parse --show-toplevel 2>/dev/null)
  if [ -n "$REPO" ] && [ -d "$REPO/.claude/skills" ]; then
    SKILLS_DIR="$REPO/.claude/skills"
  fi
fi

if [ -z "$SKILLS_DIR" ]; then
  echo "worktwin does not seem to be installed on this machine."
  echo "see https://github.com/rhagee/worktwin"
  exit 0
fi

print_skill() {
  file="$1/SKILL.md"
  [ -f "$file" ] || return
  name=$(basename "$1")
  hint=$(grep -m1 '^argument-hint:' "$file" | sed -e 's/^argument-hint: *//' -e 's/^"//' -e 's/"$//')
  desc=$(grep -m1 '^description:' "$file" | sed 's/^description: *//')
  short=$(echo "$desc" | sed 's/\. [A-Z].*/./')
  if [ -n "$hint" ]; then
    echo "/$name $hint"
  else
    echo "/$name"
  fi
  echo "  $short"
  echo ""
}

ORDER="worktwin worktwin-status worktwin-ship worktwin-ship-all worktwin-finalize worktwin-help"

for name in $ORDER; do
  if [ -d "$SKILLS_DIR/$name" ]; then
    print_skill "$SKILLS_DIR/$name"
  fi
done

for dir in "$SKILLS_DIR"/worktwin*/; do
  name=$(basename "$dir")
  case " $ORDER " in
    *" $name "*) ;;
    *) print_skill "$dir" ;;
  esac
done

echo "docs: https://github.com/rhagee/worktwin"
```
